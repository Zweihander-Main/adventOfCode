# Advent of Code

Advent of Code puzzles. Currently mostly in TypeScript with other languages anticipated in the future.

## Progress

| Year | Completed |
| :--- | --------: |
| 2015 |    7 / 25 |
| 2020 |    5 / 25 |

## Structure

```
Language
	Year
		Day
			input.txt
			p1-2.ext (for both parts)
			OR
			p1.ext
			p2.ext
```

## Instructions

#### For Typescript:

```
npm install
npm run ts-run $.ts
```

where `$.ts` is the location relative to root ie `.\advent_ts\2020\day5\p1-2.ts`
